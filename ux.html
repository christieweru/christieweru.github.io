My Projects




Integration System

I designed an interactive integration system for the UCI Rocket Project Liquids team that streamlines how engineers access and understand each part of the rocket. Built in Figma, this system enhances onboarding and makes the learning process faster and more intuitive for new team members.

Kidz Docs

Kidz Docs is a redesigned version of the popular ‘Google Docs’ but for elementary & middle school children. My group and I were tasked to make Google Docs more useable for children 12 and under while maintaining some of Google Docs core functionality. During this project I took on the role as the UX Designer and created low-fidelity mock-ups that brought our idea to life.
